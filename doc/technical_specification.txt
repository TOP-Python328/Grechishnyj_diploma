1. Общие положения
    Данные о договоре:
    - номер договора
    - место заключения (город) 
    - дата договора
    
    Данные о Застройщике: (реквизиты компании Застройщика)
    - полное наименование застройщика
    - сокращенное наименование застройщика
    - представитель застройщика (либо руководитель)
    - документ полномочий представителя
    
    Данные о Клиенте(ах):
    * Покупателей по одному договору может быть несколько
    * Покупателями могут быть физические и юридические лица
    Для физических лиц
        - фамилия, имя, отчество, пол, дата рождения, 
        - паспортные данные, серия, номер, место рождения, выдан дата
        - данные о подразделении выдавшем паспорт, выдан кем, код подразделения
        - адрес регистрации, страна, регион, район, населенный пункт, улица, дом
    Для юридических лиц
        - полное наименование, сокращенное наименование
        - ИНН, ОГРН, КПП
        - представитель застройщика (либо руководитель)
        - документ полномочий представителя
        - адрес, страна, регион, район, населенный пункт, улица, дом
    
2. Данные об Объекте
    Данные об объекте строительства:
        - наименование объекта
        - вид объекта
        - назначение объекта
        - этажность
        - общая площадь
        - материал наружных стен
        - материал перекрытий
        - класс энергоэффективности
        - сейсмостойкость
        - регион
        - город
        - улица
        - дом
    
    Данные о Земельном участке:
        - ЗУ кадастровый номер
        - площадь участка
        - вид разрешенного использования
        - регион
        - город
        - улица
        - договор аренды ЗУ 
            номер договора, дата договора, 
            номер регистрации аренды, дата регистрации аренды
    
    Данные о Разрешении на строительство
        - номер разрешения
        - дата выдачи
        - срок действия
        - сайт Застройщика

    Данные о Банке (Эскроу агент) 
        - Наименование Банка
        - ОГРН
        - ИНН
        - Лицензия №
        - Лицензия Дата

3. Предмет договора</h3>
    Данные об объекте недвижимости (квартире)
        - Номер (проектный)
        - Количество комнат
        - Площадь комнат
        - Этаж
        - Секция
        - Назначение
        - Дата передачи
        - Сайт Застройщика

4. Цена договора и порядок расчетов
    Расчет суммы
        - Сумма
        - Сумма прописью
        - Собственные средства
        - Кредитные средства
        - Номер кредитного договора
        - Дата кредитного договора
        - Город Кредитного договора
        - Отделение
        - Наименование банка
        - Организационно-правовая форма банка
        - Страна, Регион, Город, Улица, Дом
        - ИНН, Р/Счет, К/Счет
    
    Эскроу агент
        - Наименование банка
        - Организационно-правовая форма банка
        - Страна, Регион, Город, Улица, Дом
        - ИНН

    Депонент
        - Данные о покупателе
        - Срок депонирования
        - Срок внесения ДС
    
    Назначение платежа
        - Номер договора
        - Дата договора

5. Обязанности и права
    Ссылка на Застройщика
        - Сайт застройщика

6. Изменение договора
    Возврат средств
        - Сумма, Сумма прописью, Р/счет, Фамилия, Имя, Отчество, 
        - Кредитный договор №, Кредитный договор Дата

7. Реквизиты сторон
    Застройщик
        - Сокращенное наименование застройщика
        - Страна, Регион, Город, Улица, Дом
        - ИНН, КПП, ОГРН
        - Р/счет
        - Представитель, Фамилия, Имя, Отчество
    
    Участник ДДУ
        - Фамилия, Имя, Отчество, Пол, Дата рождения
        - Место рождения, Паспорт серия, Паспорт номер, Выдан дата
        - Выдан кем, Код подразделения
        - Страна, Регион, Город, Улица, Дом, Квартира

│ ХРАНЕНИЕ ДАННЫХ И ПОДКЛЮЧЕНИЕ К БД │
╰────────────────────────────────────╯

Конфигурационный файл:
	- содержит сведения о базах данных которые испльзуются в приложении

Адаптер:
	- загружет базы данных из конфигурационного файла в системные настройки при запуске сервера
	- записывает базу данных в конфигурационный файл
	- добавляет новую базу данных при регистрации пользователя в системные настройки

База данных:
    - содержит данные зарегистрированных пользователей
    - в момент регистрации (по шаблону) создается база данных для пользователя

Файловая база данных пользователя:
    - содержит данные об объектах строительства (микрорайоны, дома)
    - содержит данные о реалиемом недвижимом имуществе
    - содержит данные о клиентах (покупателях)
    - содержит данные о совершенных сделках
    - пользователь имеет доступ только к своей базе данных

Шаблон договора:
    - пользователю предоставляется базовый шаблон договора
    - пользователь имеет возможность редактировать шаблон с последующим использованием 
	
Недвижимость:
	- добавить возможность пополнять список комнат
	- добавить возможность создавать планировку квартир
	- добавить возможность создавать типовой план этажа
	- добавить возможность создавать типовую секцию
	- добавить возможность формировать многоквартирный дом с квартирами
	- добавить возможность создавать разрешение на строительство
	- добавить возможность создавать микрорайон

Адрес:
    - страна (только РФ)
    - регион
    - район
    - город
    - район города
    - населенный пункт
    - улица
    - дом
    - квартира

Регистрация
    - Показать форму регистрации
    - Проверка валидности данных 
    - Добавление нового пользователя в базу данных
    - Создание пользовательской базы данных

Авторизация
    - Показать форму авторизации
    - Проверка доступа к данным
    - Подключение к пользовательской базе данных

Главная страница
    - Отображение статистических данных 

Недвижимость
	- После рагистрации пользователь создает планировки квартир из комнат
	- Далее пользователь создает типовые этажи и секции 
	- Перед добавлением жилого дома создается разрешение на строительство и микрорайон 
	- В момент создания жилого дома имеющиеся в нем квартиры создаются автоматически на основании типовых элементов
	
Продажи
    - Формирование нового договора по шаблону
    - Добавление записей связанных со сделкой
